# E‑commerce Catalog API

A professional Node.js + Express backend project for managing a product catalog.

## Features
- REST API
- Clean folder architecture
- Controllers / Routes / Services separation
- Ready for GitHub portfolio

## Project Structure

src/
  controllers/
  routes/
  services/
  models/
  middleware/
  config/

## Install

npm install

## Run

npm start

Server runs on:

http://localhost:5000/api/products
